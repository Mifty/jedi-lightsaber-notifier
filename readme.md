Jedi Lightsaber Notifier is a tutorial from http://miftyisbored.com providing a simple introduction to notifications in Laravel 5.3. Its a simple application that sends email notifications to a database of Jedis about the status of their lightsaber. Its a basic application but presents the important aspects of notifications in Laravel.
